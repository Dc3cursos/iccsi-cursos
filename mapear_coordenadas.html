📋 Instrucciones para Mapear Coordenadas:

  1. Abre tu plantilla PDF en VS Code o tu lector de PDF favorito
  2. Identifica las posiciones exactas donde debe ir cada campo
  3. Usa una herramienta de medición o estima las coordenadas (X, Y)
  4. Llena los valores en el formulario de abajo
  5. Genera el PDF de prueba para verificar las posiciones

💡 Tip: Las coordenadas van de (0,0) en la esquina inferior izquierda hasta (612,792) en la esquina superior derecha para una página Letter.
